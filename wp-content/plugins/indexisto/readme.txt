Site search functionality for Wordpress powered sites. The plugin is based on powerful hosted ElasticSearch engine (hosted for free)

== Description ==
***This plugin allows you to create powerful search functionality for your Wordpress site***

* AJAX based instant search box (search as you type)
* Easy customization. Simple drop down search box or full screen search overlay. Templates and CSS are editable.
* Fast content indexation. Full content indexation upon install, and hooks for adding, editing.
* Rich snippets with image, text snippet, tags, date.
* Full text search server includes stemming, morphology, synonyms, fuzzy search functionality.
* Facets filtering.
* Speed: less than 70ms response.

== Installation ==
1. Upload \"indexisto\" to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. Go to \"Settings->Indexisto\" in admin panel to make initial setup.
